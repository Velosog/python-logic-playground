# Python Logic Playground

Repositório de prática em Python com foco em **fundamentos**, **funções**, **estruturas de controle** e **arquivos** — organizado por módulos e com **testes automatizados (pytest)**.

## Estrutura

- `src/fundamentos/` — listas, dicts, lógica básica
- `src/funcoes/` — funções utilitárias com validação
- `src/controle/` — if/loops/menus e validação de input (em construção)
- `src/arquivos/` — leitura/escrita (em construção)
- `tests/` — testes com pytest
